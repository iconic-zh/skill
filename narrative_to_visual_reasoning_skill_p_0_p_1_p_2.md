# Skill: Narrative-to-Visual Reasoning (P0 / P1 / P2)

> 目标：让 AI 在**无旁白/弱对白**条件下，能够像人类导演一样，从剧本中**理解因果、组织镜头、生成可被看懂的画面序列**。

本 Skill 综合：

- 认知叙事（因果 / 状态变化）
- 微电影导演方法（调度、镜头、动作、证据）
- 多轨道生成（结构先行，美学后置）

---

## 0. 核心原则（不可违背）

1. **先因果，后画面**：任何画面都必须服务于一个明确的因果节点。
2. **状态变化是剧情最小单位**：不是“发生了什么”，而是“发生后角色变成了什么”。
3. **动词优先于名词**：动作 > 场景 > 风格。
4. **观众推断优先**：如果观众无法仅凭画面推断剧情，该画面无效。
5. **多轨道但强锁序**：P0 锁顺序，P1 解释因果，P2 才负责表达。

---

## 1. 语义分级体系（Semantic Priority Levels）

### 🟥 P0 — 因果主干层（Narrative Causality Layer）

**定义**： 构成故事“因为 A → 所以 B”的**不可删减状态链**。

**P0 的本质**：

- 不是文学情节
- 不是分镜
- 是一个**角色状态机（State Machine）**

#### P0 节点判定标准（必须同时满足）：

- 绑定到【具体角色】
- 发生【心理 / 认知 / 目标 / 关系】的变化
- 存在【明确触发原因】

#### P0 标准结构：

```
P0-x:
- 角色：
- 原状态：
- 触发原因：
- 新状态：
```

#### 示例：

```
P0-3:
- 角色：A
- 原状态：信任B
- 触发原因：看到B与敌方通话的证据
- 新状态：开始怀疑B
```

> ⚠️ 若删除该节点，后续行为将“无法被理解”，则它必为 P0。

---

### 🟧 P1 — 视觉推断层（Visual Inference Layer）

**定义**： 为使观众“看懂 P0”，**必须被显性呈现的视觉信息集合**。

> P1 是“导演在片场真正关心的层级”。

#### P1 的四类必选要素：

1. **行为锚点（Action Anchor）**

   - 停下、靠近、后退、遮挡、抢夺、回避视线

2. **因果证据（Evidence / Trigger）**

   - 物件、信息、他人行为留下的痕迹

3. **情绪外化（Externalized Emotion）**

   - 表情变化、身体姿态、动作节奏

4. **时空连续性提示（Continuity）**

   - 同一空间前后对照 / 时间跳变标记

#### P1 输出模板：

```
P1-x（对应 P0-x）：
- 可观察行为：
- 关键证据：
- 情绪外化方式：
- 空间 / 时间提示：
```

> ⚠️ 若观众无法从 P1 推断出 P0，则 P1 不合格。

---

### 🟨 P2 — 表现与风格层（Expression & Aesthetic Layer）

**定义**： 在 P0 与 P1 稳定成立后，用于**强化情绪与主题感受**的表达层。

#### P2 包含：

- 构图、景别、镜头运动
- 光影、色彩、节奏
- 风格化隐喻（但不可新增剧情信息）

> ⚠️ 禁止 P2 反向决定 P1 / P0。

---

## 2. 多轨道叙事生成模型（Multi-Track Narrative Model）

```
Track A（P0）— 因果 / 状态轨（锁顺序）
Track B（P1）— 视觉推断轨（解释因果）
Track C（P2）— 风格表达轨（增强感受）
```

**生成顺序约束**：

1. 必须先完整生成 Track A
2. Track B 逐一对齐 Track A
3. Track C 不得改变 A / B 的语义

---

## 3. AI 执行流程（Skill Pipeline）

### Step 1：剧本 → P0 解析（叙事理解）

**指令**：

> 忽略画面、美学、台词修辞，只关注“角色状态如何变化”。

**输出**：P0 节点列表（严格因果）。

---

### Step 2：P0 → P1 补齐（导演化理解）

**指令**：

> 假设观众听不到任何解释，只能看画面。

**输出**：每个 P0 对应一组最少但充分的视觉线索。

---

### Step 3：P1 → 画面 / 片段生成

**硬性约束**：

- 不得新增剧情信息
- 不得跳过 P0 顺序
- 一个画面只能服务一个 P0

---

## 4. 质量自检（AI 必须自问的问题）

在生成每一个画面前，AI 必须回答：

1. 这个画面对应哪个 P0？
2. 观众能从中推断出什么变化？
3. 如果删除上一画面，这一画面还能被理解吗？
4. 是否引入了未在 P0 中定义的新信息？

若任一问题无法回答 → 画面无效。

---

## 5. Skill 使用边界

- 本 Skill **不负责**：文学润色、对白美化
- 本 Skill **专注于**：无旁白可视化叙事
- 适用于：短剧 / 微电影 / AI 分镜 / 多模态生成

---

## 6. 一句话总结（给 AI 的 Meta 指令）

> **不要生成“看起来像剧情的画面”，而要生成“能让观众推断出剧情的证据序列”。**

---

End of Skill
