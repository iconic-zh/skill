# 剧本转镜头提示词生成器

本技能把剧本文本解析为可直接用于分镜设计或AI图像/视频生成的“连贯镜头提示词”。它注重镜头之间的连续性（人物、空间、动作与光线的延续与变换），生成清晰的镜头序列与简洁可用的提示行。

## 适用场景
- 将文字剧本快速转化为分镜表或视觉参考
- 产出适配 AI 画像/视频工具（如文生图、文生视频）的提示词
- 需要保证镜头间叙事与视觉连续性时

## 输入要求
- 输入：完整剧本或片段（支持中英双语）；可包含场次标头、场景描述、人物对话与动作说明
- 可选标注：场景标头（如：INT/EXT、地点、时段），有助于更准确地设定光线与环境
- 长度：建议每次不超过约 2-3 场（超长文本可分段多次生成以保持质量与一致性）

## 输出结构
为每个镜头输出结构化信息与一行可直接使用的提示行：
- 基本信息：Scene、Shot#、镜头类型（广角/中景/近景/特写…）、构图/视角（正面/侧面/俯拍/仰拍）、运动（推/拉/摇/移/跟…）
- 主体要素：人物/对象、动作、表情/情绪、服装/道具关键点
- 环境与时间：地点、内/外景、昼/夜、天气、氛围
- 视觉风格：光线（硬/软、主光方向）、色调、风格参考（如写实/胶片/赛博/古风…）
- 连贯性标注：与上一个镜头的关系（人物位置/视线/动作延续、轴线与方向、色温/光比变化）
- 输出提示行：将上述要点串联成一行“简洁提示词”（逗号分隔，面向图像/视频模型）

示例（节选）：

```
Scene 1
Shot 1  类型: 中景  视角: 正面  运动: 缓慢推进
主体: 年轻编剧在桌前伏案  情绪: 专注略焦虑
环境: 室内, 夜, 台灯为主光, 背景书架虚化
风格: 写实, 暖色主调, 颗粒轻微
连贯性: 下一镜头承接手部动作并保留暖光色温
提示: young screenwriter at desk, night interior, warm desk lamp key light, shallow depth of field, realistic, slight film grain, subtle push-in, focused expression
```

## 生成原则
- 连贯优先：保持人物朝向/位置/动作与灯光色温的合理延续；必要时在镜头切换给出平滑过渡策略
- 信息浓缩：提示行只保留可视关键字，避免冗余与抽象空词
- 叙事明确：镜头序列应服务事件推进与情绪发展
- 轴线一致：避免不必要的跳轴；如需跨轴，先以过渡镜头或运动解决

## 可调参数（通过自然语言声明）
- 风格模板：写实/电影胶片/复古/黑色电影/赛博朋克/国风等
- 细节密度：精简/标准/详尽（影响每镜头信息颗粒度与提示行长度）
- 节奏：慢/中/快（影响镜头时长建议与运动力度）
- 输出格式：纯文本（默认）/ JSON（结构化对接下游工具）

示例声明：“使用电影胶片风格，输出 JSON，细节标准，节奏偏慢。”

## 使用指南
1) 提供剧本文本（可按场次分段）  
2) 如有偏好，附上风格/节奏/格式需求  
3) 调用后将得到镜头序列与各镜头提示行；如需多段统一风格，保持相同参数并在后续请求中强调“延续前文风格与设定”

## JSON 输出示例（节选）

```json
    }

```

## 何时调用本技能
- 用户提供剧本文本并希望生成镜头分解或 AI 提示词
- 用户希望保证镜头连续性而非仅关键词罗列
- 用户需要结构化结果对接制作流程/工具

## 限制与注意
- 不生成或放大敏感/违规内容；人物隐私与公共安全优先
- 对超长剧本建议分段处理以保证质量与连贯设定可控

## 快速开始
将你的剧本文本粘贴并附上偏好，例如：

```
请将以下片段转为连贯镜头提示词，风格写实，节奏偏慢，输出文本：
（剧本文本…）
```
